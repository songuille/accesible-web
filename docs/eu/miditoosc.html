MIDItoOSC

Midas M32 / Behringer X32 MIDI datuak OSC agindu bihurtzen ditu.

Ezaugarri nagusiak

  • MIDI iturria eta mahai-profila aukeratzea (Auto, M32/X32, X Air, Yamaha DM7/DM3/Rivage, WING, DiGiCo SD/Quantum).
  • Sarrerako MIDIa OSC garbi bihurtzea lerro bakarrean (ibilbidea + argumentuak), kopiatu, arrastatu edo bidaltzeko prest.
  • QLab-era bidalketa lerroz lerro edo sortatan (hautatutako lerroak, taldekatuta edo gabe), Egonkor/Azkar moduekin.
  • Aukerako MIDI ikaskuntza (24 urratseko kalibrazioa) eta alboko panelaren doikuntza fina detektatutako fader eta mute botoientzat.

Ikuskizunak azkar programatzeko egina: MIDI/OSC monitore argia, teklatuko lasterbideak eta kontsolaren profil errealera egokitutako bideratzea.
