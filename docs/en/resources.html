Resources

Documentation links and practical setup recommendations.

Quick setup checklist

  • Assign fixed IPs for projectors and mixer devices where possible.
  • Keep OSC ports documented per machine and app.
  • Prepare fallback manual actions for critical cues.
  • Test network behavior under show-time load.

Recommended workflow

Build cues first in QLab/your control software, then map device-level actions in each app and validate in rehearsal.
