MIDItoOSC

Translate MIDI data from Midas M32 / Behringer X32 into OSC commands for show control.

Apps

  • Read mixer parameters like faders and mutes.
  • Generate easy OSC commands to copy or drag into QLab.
  • Speed up programming for automated mix cues.
  • Architecture ready to expand to more mixer models.

More details are continuously updated as each app evolves.
