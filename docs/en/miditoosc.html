MIDItoOSC

More powerful and flexible MIDI→OSC for show control — now with Live mode to mirror or translate between mixers in real time.

Version 2.0.3 is more powerful and flexible than ever — with independent input and output profiles, smarter cross-desk routing, and a new Live mode to mirror or translate your mix to a second desk in real time.

What's new in 2.0.3

Live to a second mixer

Mirror your show in real time: send the same control stream to another desk while you work. Use matching profiles for a straight clone, or translate between different consoles (e.g. M32 → DiGiCo, WING → Yamaha) so QLab and live output stay in sync without duplicate cues.

More flexible input & output profiles

Choose input and output profiles independently — same desk end-to-end, or mix and match (e.g. control from an M32, send QLab cues as DiGiCo, live to a Roland). Auto still follows your input profile when you want a simple setup.

Smarter cross-desk routing

Broader support for faders, mutes and pan across X32/M32, X-Air, WING, DiGiCo, Yamaha DM7/DM3/Rivage and Roland — for both QLab and live output. Routes follow documented mappings; unsupported controls are blocked instead of sending wrong OSC.

DiGiCo (QLab)

Aligned with verified QLab templates: channel faders, mutes, pan, DCA, main (Multis) and group mutes — including the {parameter} path format.

Roland

M-200i and M-400 outputs use MIDI only (no accidental OSC). Live routing normalises to the M32 hub before conversion.

Learning mode

Moved to the top menu for quicker access when calibrating a new desk.

Interface

Redesigned sidebar fader — taller, vertical cap with grip lines, closer to a real desk channel fader.

Under the hood

Yamaha and live-routing fixes, stricter OSC encoding, and broader automated checks across profile combinations.

Key features

  • Live mode: clone or translate your mix to a second desk in real time — same console or different brands.
  • Independent MIDI input and output profiles (Auto, M32/X32, X Air, Yamaha DM7/DM3/Rivage, WING, DiGiCo SD/Quantum, Roland).
  • Convert incoming MIDI to clean one-line OSC (path + arguments), ready to copy, drag, or send to QLab.
  • Send to QLab per row or in batches (selected lines, grouped or ungrouped) with Stable/Fast batch modes.
  • Optional MIDI Learn (24-step calibration) plus sidebar fine editing; in OSC source mode it recognizes any mixer parameter (pan, EQ, compressor, etc.).

Built for fast show programming: clear MIDI/OSC monitor, keyboard shortcuts, and profile-aware routing for real console workflows.

Profile compatibility: MIDItoOSC is production-validated with Midas M32 / Behringer X32 and Behringer WING. Additional profiles (X Air, Yamaha DM7/DM3/Rivage, DiGiCo SD/Quantum and Roland) are provided as extended compatibility and may require adjustment depending on firmware or system configuration. For these setups, use MIDI Learn (guided 24-step calibration) to confirm and adapt the profile to your console.
