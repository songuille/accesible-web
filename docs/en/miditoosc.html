MIDItoOSC

Translate MIDI data from Midas M32 / Behringer X32 into OSC commands for show control.

Key features

  • Select MIDI source and mixer profile (Auto, M32/X32, X Air, Yamaha DM7/DM3/Rivage, WING, DiGiCo SD/Quantum).
  • Convert incoming MIDI to clean one-line OSC text (path + arguments), ready to copy, drag, or send.
  • Send to QLab per row or in batches (selected lines, grouped or ungrouped) with Stable/Fast batch modes.
  • Optional MIDI Learn (24-step calibration) plus sidebar fine editing for detected controls; in OSC source mode it recognizes any mixer parameter (pan, EQ, compressor, etc.).

Built for fast show programming: clear MIDI/OSC monitor, keyboard shortcuts, and profile-aware routing for real console workflows.

Profile compatibility: MIDItoOSC is production-validated with Midas M32 / Behringer X32 and Behringer WING. Additional profiles (X Air, Yamaha DM7/DM3/Rivage, and DiGiCo SD/Quantum) are provided as extended compatibility and may require adjustment depending on firmware or system configuration. For these setups, use MIDI Learn (guided 24-step calibration) to confirm and adapt the profile to your console.
