Preguntes frequents

Respostes rapides sobre compatibilitat, xarxa i automatitzacio de shows.

Do all apps require QLab?

No. QLab is common in theatre workflows, but apps can be used with any compatible OSC/PJLink/MIDI workflow.

Can I use several apps at once?

Yes. Typical setups combine accessibility, projection and mixer automation in the same production.

Which platforms are covered?

Current lineup includes macOS and iOS. More integrations can be added over time.
