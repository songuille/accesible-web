Privacidad

Resumen de política para productos Stage Apps.

Alcance

Esta Política de Privacidad se aplica a los productos Stage Apps, incluyendo Accesible, Shutter PJ-OSC, Shutter PJLINK y MIDItoOSC.

Tratamiento de datos

Los proyectos y la configuración de la app se procesan y almacenan principalmente en el dispositivo. No exigimos crear una cuenta de usuario para usar nuestras apps.

Uso de red local

Algunas funciones se comunican con dispositivos y servicios de red local configurados por el usuario (por ejemplo, endpoints OSC y PJLINK). Ese tráfico se usa solo para ofrecer la funcionalidad solicitada por el usuario.

Analítica y seguimiento

Los productos Stage Apps no incluyen SDK publicitarios de terceros ni realizan seguimiento entre apps o sitios web con fines publicitarios.

Suscripciones y facturación

Cuando existen suscripciones, la facturación, el cobro y la verificación del estado de suscripción los gestiona Apple mediante StoreKit. No recibimos ni almacenamos contraseñas de Apple ID ni datos completos de tarjeta.

Conservación de datos

Los datos permanecen bajo control del usuario en su dispositivo salvo que el usuario los exporte o comparta de forma explícita.
