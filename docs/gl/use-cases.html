Casos de uso

Fluxos reais combinando proxeccion, accesibilidade e automatizacion de mixer.

Case 1

Use QLab cues to trigger subtitles in Accesible and projector shutter actions in Shutter PJ-OSC at scene changes.

Case 2

Use MIDItoOSC to generate OSC commands from mixer moves and automate audio snapshots in show software.

Case 3

Use Shutter PJLINK on iOS for manual override while FOH keeps OSC automation running on macOS.
