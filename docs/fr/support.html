Assistance

Aide, prérequis et questions fréquentes

Documentation dans l’app

La référence la plus complète se trouve dans l’application : menu Aide → Documentation (fenêtre avec des sections sur les projets, QLab, OSC, sous-titres, traduction, audio et vidéo).

Prérequis

  • macOS (version minimale indiquée sur le Mac App Store).
  • Réseau local pour que les clients (navigateurs sur d’autres appareils) atteignent les serveurs HTTP/WebSocket.
  • Pour certaines fonctions (p. ex. traduction), une connexion internet peut être nécessaire sur le Mac qui exécute l’app.
  • QLab (Figure 53) lorsque vous souhaitez aligner dossiers et flux avec un projet .qlab4 / .qlab5.

Abonnement

L’utilisation complète d’Accesible nécessite un abonnement géré par Apple (StoreKit). L’achat, le renouvellement et l’annulation passent par votre compte Apple ; l’app vérifie l’état de l’abonnement sur l’appareil.

Problèmes courants

  • Les serveurs audio/vidéo ne démarrent pas : vérifiez qu’un projet est ouvert ou créé et que les dossiers du projet sont accessibles (y compris les permissions du bac à sable sur macOS).
  • Les clients ne voient pas la page : même réseau Wi‑Fi/Ethernet ; utilisez l’IP affichée par l’app, pas localhost depuis un autre appareil.
  • L’OSC n’arrive pas : port UDP correct (p. ex. 3333 selon la fonction) et pare-feu système.
